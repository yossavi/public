blocks

explaining all extension blocks

Face Exist?

return true when detects a face in front of the PC.

[x position] of [Face joint]

return the position value in the requested axis of the specific joint: Nose, Chin, left or right eye, etc.

Face expression [expression]?

return true when detects a facial expression, such as: Smile, Tounge out, mouth open or winking.

Hand Exist?

return true when detects a hand in front of the PC.

[x position] of [Hand] [Joint]

return the position value in the requested axis of the specific joint in the selected hand (left or right).

[Hand] gesture [gesture]?

return true when detects a hand gesture like: thumb up, v sign, spread out, fist and more.
